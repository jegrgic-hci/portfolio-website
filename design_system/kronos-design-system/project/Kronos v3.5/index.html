04

Motion · named durations

Fast and mechanical. Single easing curve (cubic-bezier(0.4,0,0.2,1)). Four named durations cover everything. Transitions stay on color, background, border-color, box-shadow — no transforms, no spring physics. prefers-reduced-motion zeros all four.

--k35-motion-quick 80ms
--k35-motion-base 150ms
--k35-motion-emphasis 240ms
--k35-motion-enter 320ms
05

Elevation · two tokens, floating UI only

Cards and buttons stay flat — borders do their work. Elevation is reserved for things that genuinely lift off the page: dropdowns, popovers, toasts, modals. Two tokens cover all of it.

--k35-elev-1

Dropdown, popover, toast, tooltip. 1px hairline + 4px / 12px soft drop. Lifts ~1 surface above the page.

--k35-elev-2

Modal, full-screen overlay. 16px / 32px wider drop. Lifts ~2 surfaces — clearly above any inline content.

06

Density · cozy & compact

Tables, lists, inputs and dropdowns read from --k35-density-*. Set data-density="compact" on any container and everything inside tightens. Buttons, cards, and modals are excluded — they don't benefit from compaction.
